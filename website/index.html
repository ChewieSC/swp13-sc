Projektvorstellung¶

Die Aufgabe des Teams ist, vorhandene Schachpartien (vom pgn-Format) nach RDF zu konvertieren.

Danach sollen verschiedene Abfragen der Daten möglich sein. Außerdem werden die Daten versuchsweise zum Aufbau einer Schachengine eingesetzt.

Verwendete Technologien:¶

  • Java (als Hauptprogrammiersprache)
  • git mit Github (als Versionsverwaltung)
  • Semantic Web Ontologie (Turtle)
  • Apache Tomcat mit Vaadin (als Web-Framework)
  • Jena Framework
  • Virtuoso (als Datenbank)
  • SPARQL (als Abfragesprache)
